###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i> The answer is'12', because JavaScript is a Dynamically typed language, that's why when it has this kind of scenario it's do the type "Coercion" which means it changes the types of either of the value and do the sum operation. In this case it changes the type of 1 to '1' and complete the string concatenation which result's '12' </i>

</p>
</details>

- D: 4

<p>

#### Answer: C,

<i>Here the callback function checking whether value inside num is truthy or falsy, incase of 0,null,undefined JS consider them falsy value. That's it find 3 truthy value so it increase the value of count to 3.</i>
